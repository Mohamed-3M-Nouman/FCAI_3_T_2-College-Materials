# 🎓 College Materials - Term 2 (مواد الترم الثاني)

مرحباً بكم في مستودع مواد الترم الثاني! يحتوي هذا المستودع على ملفات، مراجع، وسكاشن المواد التالية:

---

### 1. 📊 نظم دعم اتخاذ القرار - Decision Support Systems
* **📝 Description:** An introduction to the design, implementation, and evaluation of computer-based systems that facilitate decision-making processes in organizations.
* **👨‍🏫 Doctor:** Dr. Ibrahim El-Awady
* **👨‍💻 TA:** Eng. Manal Mahmoud

### 2. 🖼️ معالجة الصور - Image Processing
* **📝 Description:** A comprehensive study of digital image manipulation, including image enhancement, restoration, analysis, and feature extraction techniques.
* **👨‍🏫 Doctor:** Dr. Rasha Mahmoud Kamel
* **👨‍💻 TA:** Eng. Mohamed Abdel Hamid

### 3. 🤖 النظم الذكية - Intelligent Systems
* **📝 Description:** Exploring the fundamental principles of artificial intelligence, search algorithms, machine learning, and their applications in solving complex problems.
* **👨‍🏫 Doctor:** Dr. Sarah Salah
* **👨‍💻 TA:** Eng. Abdelrahman Gamal

### 4. 💻 نظم التشغيل - Operating Systems
* **📝 Description:** Understanding the core concepts and architecture of operating systems, including memory management, process scheduling, concurrency, and file systems.
* **👨‍🏫 Doctor:** Dr. Abdelrahman Hosny
* **👨‍💻 TA:** Eng. Ahmed Hesham

### 5. 🏗️ تطوير البرمجيات والممارسة المهنية - Software Development and Professional Practice
* **📝 Description:** Covering the software engineering lifecycle, agile methodologies, project management, and the ethical/professional responsibilities of software developers.
* **👨‍🏫 Doctor:** Dr. Rasha Mahmoud Kamel
* **👨‍💻 TA:** Eng. Aya Nassar

### 6. 🌐 البرمجة العنكبوتية - Web Programming
* **📝 Description:** Learning the modern technologies, languages, and frameworks used to build dynamic, interactive, and responsive web applications (Front-end & Back-end).
* **👨‍🏫 Doctor:** Dr. Ali Hussein & Dr. Abram Kamal William
* **👨‍💻 TA:** Eng. Asaad Nasrallah
